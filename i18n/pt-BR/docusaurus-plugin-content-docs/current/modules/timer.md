# Timer

Controle o tempo com precisão no seu código, ele contém vários métodos de temporização.

## Métodos

### new

```js
const timer = Timer.new();
```

### getTime

```js
Timer.getTime(timer);
```

### setTime

```js
Timer.setTime(src, value);
```

### destroy

```js
Timer.destroy(timer);
```

### pause

```js
Timer.pause(timer);
```

### resume

```js
Timer.resume(timer);
```

### reset

```js
Timer.reset(timer);
```

### isPlaying

```js
Timer.isPlaying(timer);
```
