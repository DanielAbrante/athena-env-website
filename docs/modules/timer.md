# Timer

Control the time precisely in your code, it contains several timing Methods.

## Methods

```js
const timer = Timer.new();
```


```js
Timer.getTime(timer);
```



```js
Timer.setTime(src, value);
```



```js
Timer.destroy(timer);
```



```js
Timer.pause(timer);
```



```js
Timer.resume(timer);
```



```js
Timer.reset(timer);
```



```js
Timer.isPlaying(timer);
```
